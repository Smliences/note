## 实例变量和类变量

实例变量就是没有static修饰的变量，有static是静态变量。

先初始化为jvm默认值，之后再初始化为你设置的值，然后执行构造方法。

static的方法无法访问非static的实例也就是没有static修饰的变量。

类的静态方法在方法区，没有实例对象，所以不在堆内存中。
